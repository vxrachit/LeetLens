# Code of Conduct

We expect all contributors and community members to follow a respectful, welcoming, and inclusive code of conduct.

Be kind and assume good intent. Harassment, discrimination, or hateful language will not be tolerated.

If you experience or witness behavior that violates this code of conduct, please open an issue and mark it for the maintainers to review, or contact the project owners directly.

This project follows the spirit of the Contributor Covenant. For serious incidents, maintainers may take any action they deem appropriate, including removing contributors or closing issues/PRs.
